# week4-task4-advanced-time-series-analysis
Time series analysis on Apple stock data using Python. Includes trend analysis and moving average visualization.
#  Apple Stock Time Series Analysis

##  Project Overview
This project focuses on performing time series analysis on Apple (AAPL) stock price data from 2020 to 2025. The goal is to understand price trends, market fluctuations, and long-term behavior using visualization and statistical techniques.



##  Dataset
The dataset was collected using the `yfinance` Python library, which provides historical stock market data.

- Stock: Apple Inc. (AAPL)
- Time Period: 2020 - 2025
- Key Feature Used: Closing Price



## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- yfinance



##  Methodology
1. Data collection using `yfinance`
2. Data cleaning and preprocessing
3. Selection of relevant features (Date, Close price)
4. Time series visualization
5. Moving average calculation (50-day window)



##  Analysis Performed
- Visualization of Apple stock closing price over time
- Trend analysis of stock movement
- 50-day moving average to smooth fluctuations
- Observation of volatility and long-term trend



##  Key Insights
- Apple stock shows an overall upward trend from 2020 to 2025.
- Short-term fluctuations indicate market volatility.
- Moving average helps identify long-term direction more clearly.
- Stock prices frequently fluctuate but follow a general growth pattern.



##  Conclusion
Time series analysis provides valuable insights into stock market behavior. This project demonstrates how historical data can be used to analyze trends and support data-driven decision making.
